Abstract

Large-scale generative language and vision-language models (LLMs and VLMs) exhibit impressive few-shot learning abilities. This capability has led to their adoption for decision making in interactive domains through in-context learning in which a prompt and domain-relevant information and examples are appended to the model's context window. However, high-quality in-domain examples are needed, and engineering those as well as the prompt itself can be costly. We propose a more efficient method to obtain prompt examples in retrieval-augmented language and vision model agents. Our method learns a memory of experience abstractions directly from raw demonstration trajectories and human feedback. Specifically, given a noisy trajectory for a task in a new domain, LLMs/VLMs can be used to abstract the trajectory into a corresponding general program of the experience. This is accomplished by removing inefficiencies, annotating subgoals, rules, exceptions, and summaries as comments in language format through appropriate prompting (e.g., ``add abstractions comments of the activity''). We demonstrate that this method rapidly learns useful experience abstractions. When used as in-context examples in retrieval-augmented LLM and VLM agents, these experiences lead to significantly improved decision-making and abstraction formation to assist future inferences. This method improves on the state-of-the-art when tested both in dialogue-based instruction following in household environments in TEACh and in multimodal autonomous web agents in Visual Web Arena. In TEACh, we improve on the state-of-the-art by 12.6% in goal-condition success, outperforming LLM agents that use the raw visual demonstrations as in context examples without abstraction learning. In Visual Web Arena, we improve on the state-of-the-art by 8.4% in overall task success, outperforming VLM agents that use hand-written examples. Lastly, we compare weight fine-tuning and in-context abstraction learning and find the two to be complementary, with a combination of the two giving the best performance.
